# Subtitle extensions

Subtitle extensions are used to search for subtitles for a specific anime episode.

They can return extensions in any known subtitle format, be it with or without file extensions, and Hayase will try to match them with the corresponding video file based on the filename and episode information.

This is especially useful for niche languages which might not have torrents subbed or dubbed in that language, but do have subtitles available. By using subtitle extensions, users can still enjoy content in their preferred language isn't available.
